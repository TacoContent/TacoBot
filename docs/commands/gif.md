# Command: gif

**Description:**
Gets a random gif from the given tag.

**Usage:**

```text
.taco gif [search_term]
```

**Arguments:**

- `search_term` (optional, string): The string to search for. Default: "taco"

**Aliases:** giphy

**Examples:**

```text
.taco gif
.taco gif taco tuesday
```

- **Admin Only:** No
